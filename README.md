# Incorporating spatial information in deep learning parameter estimation
Code of publication 'Incorporating spatial information in deep learning parameter estimation with application to the intravoxel incoherent motion model in diffusion-weighted MRI' and 'A Comparative Study of IVIM-MRI Fitting Techniques in Glioma Grading: Conventional, Bayesian, and Voxel-wise and Spatially-Aware Deep Learning Approaches'.

This code will be publicly available after the publication of both  manuscripts. 

This will include code for the supervised training on larger receptive fields utilizing neighborhood-attention, as well as the code for generating fractal-noise parameter maps.

The training code for the SA-17 model, training on both the “random-uniform” and “random-gaussian” patches, will be made publicly available upon publication of both manuscripts.
